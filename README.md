## ✨ Highlights

* Built an autonomous multi-agent system during **Hackapizza 2.0 — Agent Edition**, a 24-hour AI hackathon focused on real-time decision-making, tool usage and persistent memory.

* Participated in **three AI hackathons**, developing working prototypes involving agentic systems, local LLMs, document intelligence and semantic retrieval.

* Contributed to the publication [*Detecting and Locating Stress in Urban Settings with Chill*](https://doi.org/10.1007/978-3-031-86370-7_2), based on an Android and Wear OS system for collecting physiological and location data.

* Reimplemented and trained a lightweight image-colourization transformer with approximately **42 million parameters** on **200,000 iNaturalist images**, using a single consumer GPU.

* Developed an on-disk information retrieval system supporting **BM25, TF-IDF, DAAT and TAAT**, evaluated on MS MARCO and TREC-DL datasets.

## 🏆 Hackathons

### 🍕 [Hackapizza 2.0 — Agent Edition](https://github.com/Gianma23/domenicAIn_hackapizza)

Built an autonomous agentic system in a five-person team during a 24-hour hackathon.

The system managed a competitive galactic restaurant through real-time event handling, inventory management, blind auctions, menu planning, pricing and customer requests, operating without human intervention.

**Focus:** AI Agents · Multi-Agent Systems · Tool Calling · Long-Term Memory

### [GDG AI Hack](https://github.com/Gianma23/GDG-AI-Hack)

Developed a privacy-first local finance copilot for extracting, organizing and analysing personal financial documents without sending sensitive information to cloud services.

**Focus:** Local LLMs · Document Intelligence · Ollama · Optimization · Security

### [SinceAI Hackathon](https://github.com/Gianma23/since-ai-hackathon)

Developed an AI system that detects spare-part requests in Microsoft Teams messages, extracts product information and retrieves matching SKUs through exact and semantic search.

**Focus:** Semantic Search · FAISS · Information Extraction · AI Agents
